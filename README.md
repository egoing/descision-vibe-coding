<a id="result-python"></a>

## `result-python` 🐍 Python을 추천합니다!

Python은 읽기 쉽고 배우기 쉬운 언어로, 데이터 분석, AI, 웹 개발, 자동화 등 거의 모든 분야에서 활용할 수 있습니다.

**시작 리소스:**
- [Python 공식 튜토리얼](https://docs.python.org/ko/3/tutorial/)
- [점프 투 파이썬](https://wikidocs.net/book/1)

<a id="result-javascript"></a>

## `result-javascript` ✨ JavaScript를 추천합니다!

JavaScript는 웹의 언어로, 프론트엔드와 백엔드(Node.js) 모두에서 사용할 수 있습니다. React Native로 모바일 앱도 만들 수 있어요.

**시작 리소스:**
- [MDN JavaScript 가이드](https://developer.mozilla.org/ko/docs/Web/JavaScript/Guide)
- [모던 JavaScript 튜토리얼](https://ko.javascript.info/)

<a id="result-java"></a>

## `result-java` ☕ Java를 추천합니다!

Java는 대규모 엔터프라이즈 서비스에서 가장 많이 쓰이는 언어입니다. 안정성과 성능이 뛰어나며 취업 시장도 넓습니다.

**시작 리소스:**
- [생활코딩 Java](https://opentutorials.org/course/1223)
- [백기선의 자바 스터디](https://github.com/whiteship/live-study)

<a id="result-csharp"></a>

## `result-csharp` 🎮 C#을 추천합니다!

C#은 Unity 게임 엔진의 공식 언어로, 3D/2D 게임 개발에 최적입니다. Windows 데스크톱 앱도 만들 수 있어요.

**시작 리소스:**
- [Unity Learn](https://learn.unity.com/)
- [Microsoft C# 문서](https://learn.microsoft.com/ko-kr/dotnet/csharp/)

<a id="result-kotlin"></a>

## `result-kotlin` 📱 Kotlin을 추천합니다!

Kotlin은 Google이 공식 지원하는 Android 개발 언어입니다. Java보다 간결하고 현대적인 문법을 가지고 있습니다.

**시작 리소스:**
- [Android 개발자 가이드](https://developer.android.com/kotlin?hl=ko)
- [Kotlin 공식 문서](https://kotlinlang.org/docs/getting-started.html)

<a id="result-swift"></a>

## `result-swift` 🍎 Swift를 추천합니다!

Swift는 Apple이 만든 iOS/macOS 앱 개발 언어입니다. 안전하고 빠르며, SwiftUI로 아름다운 UI를 쉽게 만들 수 있습니다.

**시작 리소스:**
- [Swift Playgrounds](https://www.apple.com/swift/playgrounds/)
- [Apple 개발자 문서](https://developer.apple.com/kr/swift/)
