## `result-python-study` 🐍 Python을 추천합니다!

Python은 읽기 쉽고 배우기 쉬운 언어로, 대학 CS 수업과 취업 시장 모두에서 가장 널리 쓰입니다.

**이 언어로 만든 유명 서비스:** 인스타그램, 유튜브 추천 알고리즘, 드롭박스

**첫 번째 미니 프로젝트:** 가위바위보 게임 만들기 — 약 30분

**다음에 배우면 좋은 것:** 웹 개발(Django/Flask) 또는 데이터 분석(pandas)

**시작 리소스:**
- [Python 공식 튜토리얼](https://docs.python.org/ko/3/tutorial/)
- [점프 투 파이썬](https://wikidocs.net/book/1)

<a id="result-python"></a>

## `result-python` 🐍 Python을 추천합니다!

Python은 읽기 쉽고 배우기 쉬운 언어로, 데이터 분석, AI, 웹 개발, 자동화 등 거의 모든 분야에서 활용할 수 있습니다.

**이 언어로 만든 유명 서비스:** 인스타그램, 유튜브 추천 알고리즘, 드롭박스

**첫 번째 미니 프로젝트:** 날씨 정보를 가져오는 스크립트 만들기 — 약 1시간

**다음에 배우면 좋은 것:** 웹 개발(Django/Flask) 또는 데이터 분석(pandas)

**시작 리소스:**
- [Python 공식 튜토리얼](https://docs.python.org/ko/3/tutorial/)
- [점프 투 파이썬](https://wikidocs.net/book/1)

<a id="result-javascript"></a>

## `result-javascript` ✨ JavaScript를 추천합니다!

JavaScript는 웹의 언어로, 브라우저에서 바로 실행되어 결과를 즉시 확인할 수 있습니다. React로 웹앱, React Native로 모바일 앱도 만들 수 있어요.

**이 언어로 만든 유명 서비스:** 페이스북, 에어비앤비, 넷플릭스 웹 화면

**첫 번째 미니 프로젝트:** 클릭하면 색이 바뀌는 버튼 만들기 — 약 20분

**다음에 배우면 좋은 것:** React (웹앱) 또는 Node.js (서버)

**시작 리소스:**
- [MDN JavaScript 가이드](https://developer.mozilla.org/ko/docs/Web/JavaScript/Guide)
- [모던 JavaScript 튜토리얼](https://ko.javascript.info/)

<a id="result-java"></a>

## `result-java` ☕ Java를 추천합니다!

Java는 대규모 엔터프라이즈 서비스에서 가장 많이 쓰이는 언어입니다. 안정성과 성능이 뛰어나며 취업 시장도 넓습니다.

**이 언어로 만든 유명 서비스:** 카카오, 네이버, 쿠팡 백엔드

**첫 번째 미니 프로젝트:** 계산기 프로그램 만들기 — 약 1시간

**다음에 배우면 좋은 것:** Spring Boot (웹 서버 프레임워크)

**시작 리소스:**
- [생활코딩 Java](https://opentutorials.org/course/1223)
- [백기선의 자바 스터디](https://github.com/whiteship/live-study)

<a id="result-r"></a>

## `result-r` 📊 R을 추천합니다!

R은 통계 분석과 데이터 시각화에 특화된 언어입니다. 연구자, 데이터 분석가에게 Python과 함께 가장 많이 쓰입니다.

**이 언어로 만든 것들:** 학술 논문 차트, 의료 데이터 분석, 선거 예측 모델

**첫 번째 미니 프로젝트:** CSV 데이터로 그래프 그리기 — 약 30분

**다음에 배우면 좋은 것:** ggplot2 (시각화), Shiny (대화형 대시보드)

**시작 리소스:**
- [R 공식 입문 가이드](https://www.r-project.org/)
- [R for Data Science (한국어)](https://r4ds.hadley.nz/)

<a id="result-csharp"></a>

## `result-csharp` 🎮 C#을 추천합니다!

C#은 Unity 게임 엔진의 공식 언어로, 3D/2D 게임 개발에 최적입니다. Windows 데스크톱 앱도 만들 수 있어요.

**이 언어로 만든 유명 게임:** 포켓몬 GO, 컵헤드, 서브노티카

**첫 번째 미니 프로젝트:** Unity로 공 굴리기 게임 만들기 — 약 2시간

**다음에 배우면 좋은 것:** Unity Shader, C# 고급 패턴

**시작 리소스:**
- [Unity Learn](https://learn.unity.com/)
- [Microsoft C# 문서](https://learn.microsoft.com/ko-kr/dotnet/csharp/)

<a id="result-kotlin"></a>

## `result-kotlin` 📱 Kotlin을 추천합니다!

Kotlin은 Google이 공식 지원하는 Android 개발 언어입니다. Java보다 간결하고 현대적인 문법을 가지고 있습니다.

**이 언어로 만든 유명 앱:** 에어비앤비 Android 앱, 트렐로 Android 앱

**첫 번째 미니 프로젝트:** 할 일 목록(Todo) 앱 만들기 — 약 2시간

**다음에 배우면 좋은 것:** Jetpack Compose (최신 Android UI)

**시작 리소스:**
- [Android 개발자 가이드](https://developer.android.com/kotlin?hl=ko)
- [Kotlin 공식 문서](https://kotlinlang.org/docs/getting-started.html)

<a id="result-swift"></a>

## `result-swift` 🍎 Swift를 추천합니다!

Swift는 Apple이 만든 iOS/macOS 앱 개발 언어입니다. 안전하고 빠르며, SwiftUI로 아름다운 UI를 쉽게 만들 수 있습니다.

**이 언어로 만든 유명 앱:** 링크드인 iOS 앱, 에버노트 iOS 앱

**첫 번째 미니 프로젝트:** 계산기 앱 만들기 — 약 2시간

**다음에 배우면 좋은 것:** SwiftUI (최신 Apple UI 프레임워크)

**시작 리소스:**
- [Swift Playgrounds](https://www.apple.com/swift/playgrounds/)
- [Apple 개발자 문서](https://developer.apple.com/kr/swift/)
